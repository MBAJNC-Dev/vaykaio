
# AI Vacation Planner - Phase 1 Baseline

This repository contains the Phase 1 baseline for the expanded AI Vacation Planner platform. It establishes the core architecture, design system, marketing skeleton, and structural readiness for advanced future modules.

## Phase 1 Deliverables
1.  **Design System:** Comprehensive Tailwind CSS configuration (`index.css`) with semantic tokens, typography (Outfit & Space Grotesk), and responsive utilities.
2.  **Marketing Skeleton:** SEO-ready, responsive page structures for Home, Features, Pricing, About, Contact, Blog, Case Studies, Testimonials, FAQ, and Auth flows.
3.  **Global Layout:** Scalable `Header.jsx` and `Footer.jsx` designed to accommodate future navigation expansion.
4.  **Advanced Module Stubs:** Placeholder components for ML, Blockchain, AR/VR, IoT, Real-Time, Personalization, Security, Analytics, and Performance Monitoring.
5.  **Marketing Asset Structure:** Directories and `BrandGuidelines.md` stubbed out for the content team.

## Future Expansion (TODOs)
Throughout the codebase, you will find `// TODO PHASE 2:` comments indicating where and how to expand.

*   **ML/AI Engine:** Integrate TensorFlow.js or external AI APIs into `MLModuleStub.jsx`.
*   **Blockchain/Web3:** Implement wallet connections and smart contract interactions in `BlockchainModuleStub.jsx`.
*   **AR/VR:** Add WebXR or Three.js implementations in `ARModuleStub.jsx` and `VRModuleStub.jsx`.
*   **IoT:** Connect to smart luggage or hotel APIs in `IoTModuleStub.jsx`.
*   **Content:** Flesh out marketing copy, blog posts, and case studies in the respective page components.
*   **SEO:** Expand `react-helmet` usage with dynamic OpenGraph and Twitter card meta tags.

## Getting Started
1.  Run `npm install` to ensure all base dependencies are loaded.
2.  Run `npm run dev` to start the Vite development server.
3.  Navigate through the marketing pages to review the structural layout.
